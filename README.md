# Rest Yo Eyes 👀

A React application built with Vite and Tailwind CSS to help you reduce eye strain by following the 20-20-20 rule: every 20 minutes, look at something 20 feet away for 20 seconds.

## Features 

*   **20-20-20 Timer**: Automatically tracks your work time and prompts you to rest your eyes.
*   **Guided Eye Exercises**: Provides guided exercises during the 20-second rest interval to maximize eye relaxation.
*   **Progress Tracking**: Tracks total eye rests completed using local storage, gamifying the experience and providing insights into your habits.
*   **Modern UI/UX**: Built with a clean, responsive interface using React and modern CSS.

## Technologies Used 🛠️

*   [React 19](https://react.dev/)
*   [TypeScript](https://www.typescriptlang.org/)
*   [Vite](https://vitejs.dev/)
*   [Tailwind CSS v4](https://tailwindcss.com/)

## Getting Started 🚀

### Prerequisites

*   Node.js (v18 or higher recommended)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/restyoeyes.git
    cd restyoeyes
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser and navigate to `http://localhost:5173`.

## Available Scripts 📜

*   `npm run dev`: Starts the development server.
*   `npm run build`: Compiles TypeScript and builds the app for production.
*   `npm run lint`: Runs ESLint to check for code quality issues.
*   `npm run preview`: Locally previews the production build.
